¿Cuál es la diferencia entre Internet y la World Wide Web?

La diferencia es que el internet es el sistema de computadoras o redes informáticas que están conectadas entre sí en todo el mundo; y la WWW es el siustema que funciona a través del internet, o sea no se puede acceder a WWW sin Internet.
La world Wide Web es el sistema en donde se gestiona la información que se comparte por internet.

    Preguntas del lab 3, css

    Como ingeniero de software ¿cuál es tu recomendación sobre el uso de !important en un CSS?

    El usar !important hace que le de importancia a algúna declaración de estilo más que a otra. Usarlo es malo ya que además de que no sigue buenas prácticas, aumenta la complejidad, es más dificil de leer y entender como código general y rompe ese equema de importancia que lleva el in line, el documento y el css. Hace de todo un show.

    Si se pone una imagen de fondo en una página HTML, ¿por qué debe escogerse con cuidado?

    Se debe de tener cuidado porque afecta en varias cosas. puede hacer:

    1. Sea más dificil de ver el contenido
    2. No lo hace un dieño adaptable
    3. Hace que el tiempo de carga sea mayor

    En general hace más complejo todo, para el sistema y para el usuario final.

    Como ingeniero de software, ¿cuál es tu recomendación al elegir las unidades de un propiedad de estilo entre %, px y pt?

    Que usen el % cuando ese elemento dependa más el tamaño de otro elemento, px cuando tengan seguro el tamaño que quieren para el elemento independientemente de cualquier otro factor y pt para cuando se quiere imprimir, porque este se basa en centímetros y px en píxeles

    ¿Por qué el uso de una versión minimizada del CSS mejora el rendimiento del sitio?

    A grande escala, porque hace todo más fácil. El archivo es mas pequeño, los tiempos son menores, el servidor usa menos recursos y entenderlo es más sencillo.
